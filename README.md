# **DantotsuRe** 🌟

DantotsuRe is an [Anilist](https://anilist.co/) only client.
> **DantotsuRe** is a complete rewrite of [Dantotsu](https://github.com/rebelonion/Dantotsu/) in flutter.

> **Dantotsu (断トツ; Dan-totsu)** literally means "the best of the best" in Japanese. Try it out for yourself and be the judge!
